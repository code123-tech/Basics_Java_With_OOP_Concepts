- **Remove RMI Activation (JEP 407)**: Removes legacy RMI Activation mechanism.
- **Remove Experimental AOT and JIT Compiler (JEP 410)**: Removes experimental `jaotc` and Graal-based JIT integration.
- **Restore Always-Strict Floating-Point Semantics (JEP 306)**: Makes floating-point behavior strictly consistent across platforms again.
- **New macOS Rendering Pipeline (JEP 382)**: Modernized Java 2D pipeline on macOS.
- **macOS/AArch64 Port (JEP 391)**: Official support for Apple Silicon (AArch64) macOS.
