##### Features in Java 12
- Updated `Switch Expressions Flow`
- `Shenandoah GC` (Experimental Low-Pause-Time Garbage Collector)
- `Microbenchmarking Suite (JMH) in the JDK`
- `JVM Constants API`
- One AArch64 Port (Linux/ARM64)
- Default CDS Archives
- Adorable Mixed Collections for G1
