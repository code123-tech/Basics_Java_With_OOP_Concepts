##### Features in Java 14
- Switch Expressions (Standardized)
- Pattern Matching for instanceof
- Records (JEP 359) (As a preview feature)
- Deprecating the Nashorn JavaScript Engine
- Helpful NullPointerExceptions
- JFR Event Streaming
- NUMA-Aware Memory Allocation for G1
- Text Blocks (JEP 368)
- Non-Volatile Mapped Byte Buffers (JEP 352)
