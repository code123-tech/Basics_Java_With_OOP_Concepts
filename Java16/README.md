##### Features in Java 16
- Vector API (Incubator) (JEP 338): Experimental feature to express vector computations
- Enable C++14 Language Features (JEP 347): source code written in C++14
- Migrate from Mercurial to Git (JEP 357)
- Migrate to GitHub (JEP 369)
- ZGC: Concurrent Thread-Stack Processing (JEP 376)
- Unix-Domain Socket Channels (JEP 380)
- Alpine Linux Port (JEP 386)
- Elastic Metaspace (JEP 387): To improve the memory space in HotSpot VM where class-metadata is stored
- Windows/AArch64 Port (JEP 388)
- Foreign Linker API (JEP 389)
- Warnings for Value-Based Classes (JEP 390)
- Packaging Tool (JEP 392)
- Foreign-Memory Access API (JEP 393)
- Pattern Matching for instanceof (JEP 394)
- Records (JEP 395): offer a way to create simple, immutable, shallowly immutable data carrier classes
- Strongly Encapsulate JDK Internals by Default (JEP 396)
- Sealed Classes (JEP 397): In Java15 it was a preview feature, now it is standardized
